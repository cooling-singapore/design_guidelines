# design_guidelines

Plots for the design guidelines of CS 1.5

## How does it work

There are three jupyter notebooks. Open each notebook and run all cells. That is all!

## Installation

- Clone this repository
- Install dependencies
  - `Pandas`
  - `numpy`
  - `matplotlib`
  - `jupyter notebook`
